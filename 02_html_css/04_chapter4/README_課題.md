### 🚨 【超・重要ルール】画像は必ず自分の好きなものに変更すること！

見本のサイトに載っている画像をそのまま使うのは**絶対にNG**です！**必ずあなた自身の画像に変更してサイトを制作してください。**

- **ヘッダー画像（1枚）**：自分を表わしてそうな画像、ぶっちゃけ自由です。
- **アイコン画像（1枚）**：自分の顔写真、お気に入りのイラスト、推しのアバターなど
- **趣味の画像（3枚）**：あなたが好きなこと（旅行、カフェ、ゲーム、アニメ、音楽など）の写真

> 💡 **「世界にひとつだけの、あなたの個性が伝わるオリジナル自己紹介サイト」を作ることがこの課題の本当のゴールです！** どんな写真やイラストを使っても自由ですので、お気に入りの素材を自分で用意してください！

### 📋 課題の具体的な内容（実装チェックリスト）

具体的にどんなサイトを作るのか、以下の要素（要件）を満たすように作ってみましょう。

#### ① ヘッダー（最上部）

- 左側にあなたのサイトのタイトル（例：`Profile` や `〇〇's Portfolio`）を置く。
- 右側にメニューリンクを2つ（`About` と `Likes`）置く。
- **【CSS要件】**：Flexboxを使って、タイトルとメニューが「左端と右端」に綺麗に離れて横並びになるように配置する。

#### ② Aboutセクション（自己紹介）

- 見出し（`About`）を作る。
- 用意した**自分のプロフィール画像を挿入し、CSSで綺麗な「丸型」に切り抜く**。
- 画像の横（または下）に、あなたの名前と自己紹介の文章（テキスト）を書く。

#### ③ Likesセクション（好きなこと・趣味）

- 見出し（`Likes`）を作る。
- **趣味のカードを「3つ」** 作る。それぞれのカードには「趣味の画像（1枚）」「趣味のタイトル」「簡単な説明文」を入れる。
- **【CSS要件】**：Flexboxを使って、3つのカードが**綺麗に横並び**になり、カード同士の間に**適切なすき間（`gap`）** が空くようにする。

### 🛠️ 制作の進め方

一気に作ろうとすると迷子になってしまうので、以下の順番で少しずつ組み立てていくのがおすすめです！

1. **【ステップ1】HTMLで「箱」をすべて用意する**

- まずは色や配置は無視して、最上部のヘッダー、自己紹介（About）、趣味（Likes）の文字や画像を上から順番にHTMLで書いていきましょう。

2. **【ステップ2】CSSで「色・サイズ・余白」を整える**

- 背景色を塗ったり、文字の大きさを変えたり、`padding` や `margin` を使って窮屈にならないように見栄えを整えます。

3. **【ステップ3】Flexboxで「横並び」にする**

- 最後に、ヘッダーのメニューや、趣味のカード（3つ）を `display: flex;` を使って綺麗に横に並べたら完成です！

### 🖼️ 画像（素材）の準備と挿入のヒント

画像を表示させるためには、ファイルの置き場所（フォルダ構成）がとても重要になります。以下の手順通りに進めてみましょう。

#### 1. フォルダの中に「images」フォルダを作ろう

VS Codeで課題のフォルダ（`04_chapter4` など）を開き、`index.html` と同じ場所に新しく **`images`** という名前のフォルダを作ります。

#### 2. 画像の名前を「半角英数字」に変えて保存しよう

用意した4枚の画像を、先ほど作った `images` フォルダの中に保存します。
このとき、画像の名前が「写真.jpg」などの日本語になっているとエラーの原因になります。必ず**半角の英語や数字**に変えておきましょう！

> 💡 **名前の例:**
>
> - プロフィール画像 ➔ `icon.jpg`
> - 趣味の画像3枚 ➔ `like1.png` , `like2.jpg` , `like3.jpg`

#### 3. HTMLに正しい「パス（置き場所）」を書こう

Chapter1で学んだ `<img>` タグを使って、以下のように記述します。
「`images` フォルダの中にある、`icon.jpg` という画像を表示してね」という指示になります。

```html
<img src="images/icon.jpg" alt="プロフィールアイコン" />

<img src="images/like1.png" alt="趣味の画像1" />
```

#### 🚨 画像が表示されない・大きすぎる！ときの対処法

- **画面に「割れた画像マーク」が出て表示されないとき**
  原因の9割はスペルミスです！
- `images` フォルダの名前が `image`（sがない）になっていませんか？
- 拡張子（`.jpg` や `.png`）まで正しく合っていますか？大文字の `.JPG` になっているケースもあるので確認してみましょう。

- **画像が巨大すぎて画面からハミ出しているとき**
  画像を入れた直後は、写真の元のサイズで表示されるため画面が崩れます。これは正常なので安心してください！
  CSS側で、以下のように横幅（`width`）を指定してあげることで、綺麗なサイズに小さく整えることができます。

```css
/* プロフィール画像を横幅150px、高さ150pxの正方形にする場合 */
.profile-img {
  width: 150px;
  height: 150px;
  object-fit: cover; /* 画像が縦横にギュッと潰れるのを防ぐおまじない */
}
```

#### 1. HTMLの骨組み（Chapter 1）

- サイト全体のタイトルや、セクションの見出しには `<h1>` や `<h2>` を使う
- 文章には `<p>` を使う

#### 2. 画像（素材）の準備とパスの通し方

1. 課題フォルダ(04_chapter4)の中に、新しく **`images`** という名前のフォルダを作ります。
2. 用意した自分の画像の名前を、必ず半角英数字（例：`icon.jpg`, `like1.png`）に変えて `images` フォルダに保存します。
3. HTMLの `<img>` タグの `src` 属性で、正しい置き場所（パス）を指定します。

```html
<img src="images/icon.jpg" alt="プロフィール" />
```

#### 3. CSSの装飾と余白（Chapter 2）

HTMLの `<head>` 内に `<link rel="stylesheet" href="style.css">` を書くのを忘れずに！

- プロフィール画像を丸く切り抜くおまじないは `border-radius: 50%;` と `object-fit: cover;`
- 箱の「内側の余白」を広げるのは `padding`、隣の箱と「外側の距離」を空けるのは `margin`

#### 4. Flexboxで横並び（Chapter 3）

横に並べたい要素の「親の箱」に命令をかけます。

- ロゴとメニューを左右の両端に離す：`display: flex; justify-content: space-between; align-items: center;`
- 3つの趣味カードを横に並べてすき間を空ける：`display: flex; gap: 20px;`

</details>
