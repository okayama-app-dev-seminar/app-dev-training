# 00.前置き

## はじめに

このチャプターでは、Webサイト制作に必須の**HTML**と**CSS**を深く学んでいきます。

最終的に、このページの下部にある「課題（自己紹介サイトの制作）」がクリアできれば、このチャプターは無事に修了（合格）となります！

今の自分の実力に合わせて、以下のどちらかのルートで進めていきましょう。

## ルート選択について

### 1. 🏃‍♂️ 【経験者ルート】HTML/CSSの基本がすでに分かる人

基礎の解説を読み飛ばして、**ページ下部の「課題」にそのまま直接チャレンジ**してOKです！
分からない部分が出てきたら、適宜カリキュラムの解説を振り返るか、オフィスにいる先輩に気軽に質問してください。

### 2. 🔰 【初心者ルート】プログラミングに初めて触れる人

まずは、**用意されているカリキュラム（解説・ステップ）を上から順番**にやっていきましょう！
手順通りに進めれば誰でも必ず完成できるようになっています。自分のペースでじっくり進めてみてください。

## 【推奨】VS Codeの拡張機能について

効率よく開発を進めるために、VS Code（コードを書くソフト）に以下の拡張機能を導入することをお勧めします。
拡張機能は以下の画像の赤枠に囲まれたところにあります！

<img width="1000" height="500" alt="VSCode-Image" src="https://github.com/user-attachments/assets/be59c0ec-a569-4a9f-a941-df19989cf32a" />

- Prettier
  - コードを整えてくれるもの
- ESLint
  - スペルミスや、文法エラーの際「赤色の波線」を引いて警告してくれる
- LiveServer
  - コードを保存した瞬間に、ブラウザの画面も自動で最新の状態に更新してくれる
- Material Icon Theme
  - ファイル左側のアイコンを分かりやすくしてくれるもの

**HTMLとCSSを書く際は「LiveServer」をよく使うので「使い方」を詳しく紹介します！**

### 💻 Live Serverの「使い方」

<img width="1000" height="500" alt="LiveServer-Image" src="https://github.com/user-attachments/assets/99ac790f-02dc-4796-bbc7-bd870ccf3e82" />

1. 自分の書いている index.html のファイルをVS Codeで開きます。

2. 画面の右下の端っこ（ステータスバー）を見てみると、**Go Live** という文字が表示されているので、ここをクリックします。

3. 自動的にブラウザが立ち上がり、自分の作ったWebサイトが表示されたら成功です！

4. 後は、コードを書き換えて保存（Ctrl + S / Cmd + S）するだけで、ブラウザが自動的にパッと切り替わります。
