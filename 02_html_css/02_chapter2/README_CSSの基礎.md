# Chapter2 CSSの基礎

HTMLで作った箱（`div`タグとか`p`タグなど）に、色やサイズを指定していく、一番変化が分かりやすいチャプターです。

本文中に【ミニワーク】と書かれている箇所では、**実際に自分でコードを書いたり編集したりして、動作を確認**してみてください！

## ■ セクション 2.1：CSSってなに？

### 1. CSSの役割

Chapter1で、HTMLはWebサイトの「骨組み（文章の構造）」だと学びました。しかし、以下の画像のように、HTMLだけでは文字が白黒で縦に並ぶだけです。

<!-- HTMLとCSSの変化が分かる画像が入る -->

そこに、色、サイズ、配置などの **「見た目（服やメイク、インテリア）」を整えるのが** CSS（シー・エス・エス）の役割です。

### 2. 基本の書き方：3つの要素

CSSを書くときは、コンピューターに正確に伝わるよう、以下の3つのセット（構文）で記述します。

- **セレクタ**：「だれに」（HTMLのどのタグやクラスに当てるか）

- **プロパティ**：「なにを」（色か、大きさか、背景か）

- **値（ち）**：「どうするか」（赤にするか、20pxにするか）

```css
/* 基本の形 */
セレクタ {
  プロパティ: 値;
}

/* 実際のコード例（h1タグの文字を赤にする） */
h1 {
  color: red;
}

/* クラス名で書くと以下のような形 (今回だと、homeというクラスに500pxの高さを付ける) */
.home {
  　height: 500px;
}
```

> ⚠️ **初心者が100%引っかかる注意点**
>
> - プロパティの後ろは **コロン** `:` です。
> - 値の後ろは **セミコロン** `;` です。 これが1つでも抜けると、それ以降のCSSがすべて動かなくなります。
> - ここでもスペースや記号はすべて**半角**で書きます。

### 3. HTMLへの読み込み方（おまじない）

CSSを有効にするには、HTMLファイルに「このCSSを読み込んでね」と教えてあげる必要があります。

1.  `index.html` と同じフォルダー内に、**`style.css`** という名前のファイルを作ります。

2.  `index.html` の `<head>` 〜 `</head>` の間に、以下の1行を書き足します。

```html
<link rel="stylesheet" href="style.css" />
```

### 4. クラス名（`class="..."`）を使いこなそう

もし `p { color: blue; }` と書くと、サイト内のすべての文章が青くなってしまいます。
「この文章だけ赤にしたい」「この箱だけ背景をつけたい」というときは、HTML側にクラス名（名前のタグ）をつけて、CSSでそれを指名します 。

- **HTML側**：`class="名前"` をつける

- **CSS側**：名前の頭に **ピリオド `.**` をつける（超重要！）

```html
<p class="intro-text">ここだけ特別な文章です</p>
```

```css
/* CSS */
.intro-text {
  color: red;
}
```

## ■ セクション 2.2：【重要】ボックスモデルをマスターする

ここがCSS最大の難所であり、一番大切な概念です。ここさえ理解すれば、表示がグチャグチャに崩れる「レイアウト迷子」から卒業できます。

### 1. すべての要素（箱）は4つの層でできている

第1章で「Webサイトはすべて四角い箱の集まり」と学びました 。
CSSの世界では、このすべての箱が**4つの要素**で構成されています 。これを**ボックスモデル**と呼びます 。

1.  **コンテンツ（中身）**：文字や画像そのものが入るエリア 。

2.  **padding（パディング / 内側の余白）**：中身と、箱の「境界線」との間にあるすき間 。

3.  **border（ボーダー / 枠線）**：箱の境界線（線そのもの） 。

4.  **margin（マージン / 外側の余白）**：箱と「その隣にある別の箱」との間にあるすき間 。

### 2. 「内側の余白」と「外側の余白」の使い分け

初心者が最も混乱するのが、`padding` と `margin` の違いです。

- **padding（内側の余白）**：**箱の内側を広げるもの** 。

- 背景色を塗ったり枠線をつけたりしたとき、文字が線にピタッとくっついていると窮屈ですよね。箱の内側にゆとりを持たせたいときに使います 。

- **margin（外側の余白）**：**隣の箱を押し返すもの** 。

- 上の箱と下の箱、右の箱と左の箱の間に「ディスタンス（距離）」を取りたいときに使います 。

### 3. 箱のサイズを決めるプロパティ

- **`width`**：箱の横幅を指定します 。

- **`height`**：箱の高さを指定します 。

```css
.box {
  width: 300px; /* 横幅を300ピクセルに */
  background-color: #f0f0f0; /* 背景をグレーに */
  padding: 20px; /* 内側に20pxのゆとりを作る */
  margin-bottom: 40px; /* 下にある別の箱を40px押し下げる */
}
```

## ■ セクション 2.3：よく使う文字と背景のデザイン

ここからは、実際に2週目の課題サイトで使うプロパティを使って、デザインをガラリと変える体験をしましょう！

### 1. これだけは外せない定番プロパティ

#### ① 文字のデザイン

- **`color`**：文字の色を変えます 。

- **`font-size`**：文字の大きさを変えます（例：`font-size: 16px;`） 。

- **`font-weight`**：文字の太さを変えます（例：太字にするなら `bold`） 。

- **`text-align`**：文字の配置（左寄せ、中央寄せ、右寄せ）を決めます 。

- 💡2週目への伏線：見出しを画面の真ん中に持ってくるために `text-align: center;` を多用します 。

#### ② 背景と角丸のデザイン

- **`background-color`**：箱の背景に色を塗ります 。

- **`border-radius`**：箱の角を丸くします 。

- 2週目の課題で、四角いプロフィール画像を**綺麗な丸型**に切り抜くおまじないは、これを使って `border-radius: 50%;` と書くことです 。

### ✍️ 【ミニワーク】自己紹介を「名刺カード」にしてみよう！

第1章で作った自己紹介HTML（`index.html`）に、以下のコードを書き足してブラウザで確認してみましょう 。ただの文字の羅列が、一気に「デザインされたカード」に化けます！

**HTML側：** （`<div>` タグにクラス名を追加します）

```html
<div class="profile-card">
  <h2>名前：山田 太郎</h2>
  <p>はじめまして！新入生の山田です。Web制作の勉強を頑張ります！</p>
  <a href="#">私のリンク</a>
</div>
```

**CSS側（`style.css`）：**

```css
/* カード全体の見た目 */
.profile-card {
  width: 400px;
  background-color: #e6f2ff; /* 薄い青色 */
  border: 2px solid #0066cc; /* 濃い青色の枠線 */
  border-radius: 10px; /* 角を丸くする */
  padding: 20px; /* 内側に余白を作る（padding） */
  margin: 50px auto; /* 画面の真ん中に配置するための margin */
}

/* カードの中の見出し */
.profile-card h2 {
  color: #0066cc;
  text-align: center; /* 文字を中央揃え */
  margin-bottom: 15px; /* 下の文章との間を空ける */
}
```
