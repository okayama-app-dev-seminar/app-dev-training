# Chapter3 CSSの超重要スキル「Flexbox」

要素を**様々な方向に動かす**ことができる「Flexbox」について学習します。

## ■ セクション 3.1：なぜ「横並び」が必要なのか？

### 1. HTML要素のデフォルトの動き（上から下へ）

Chapter1とChapter2で、HTMLを使って箱を作り、CSSで色を塗る方法を学びました。
しかし、これまでに書いたコードを思い出すと、すべての要素（見出し、文章、画像）は**自動的に「上から下へ」と縦一列に積み重なっていた**はずです。

これは、HTMLの要素（特に `<div>` や `<p>`、`<h2>` など）が、標準のルールとして「画面の横幅を1行まるまる占領する」という性質を持っているからです。

### 2. 課題の内容を観察してみよう

ここで、Chapter4で皆さんが作る課題を見てみましょう。

- 画面最上部のヘッダー：左側に「Profile」、右側に「About / Likes」が**横に並んでいます**。
- Likes（趣味）のコーナー：3つの趣味カードが**横に3つ並んでいます**。

世の中のほぼ全てのWebサイトは、要素を横に並べることで読みやすく美しいレイアウトを実現しています。この「縦に並ぶはずの要素を、自由自在に横に並べる魔法」こそが、今回学ぶ **Flexbox（フレックスボックス）** です。

#### ② 横方向の並び方を変える（動画 05:26〜）


#### ③ 縦方向の揃え方を変える（動画 06:55〜）

動画の **【6分55秒】** に飛びます。上下の配置を揃える `align-items` の実験です。
※今回は分かりやすいように、親の箱の高さ（`height: 300px;`）と枠線（`border`）を一時的に足して試してみましょう。

```css
.container {
  display: flex;
  height: 300px; /* 高さをわざと広げます */
  border: 2px solid #333; /* 親の箱の大きさをわかりやすくします */

  align-items: center; /* ★上下の真ん中に揃える命令 */
}
```

👉 **確認してみよう**：子要素が、高さ300pxの箱の中央（上下の真ん中）に綺麗に浮いて並びます！ [[08:00](http://www.youtube.com/watch?v=UF5SDgT2eTc&t=480)]

<details>
<summary>🔍 もっと詳しく知りたいときのヒント（トグルを開いてね）</summary>

Flexboxを使う上で、初心者が一番間違えやすい超・重要ルールです。
それは、**「横並びにしたい要素（子要素）ではなく、それらを包んでいる箱（親要素）に命令を書く」** ということです。

例えば、3つの趣味カード（子）を横に並べたいときは、その3つのカードを囲んでいる「カードたちのまとめ役の箱（親）」に対してCSSを書きます。

```html
<div class="likes-container">
  <div class="like-card">趣味1</div>
  <div class="like-card">趣味2</div>
  <div class="like-card">趣味3</div>
</div>
```

```css
.likes-container {
  display: flex; /* これを書くだけで、中のカードが横に並びます！ */
}
```

親の箱に `display: flex;` を書くと要素は横に並びますが、デフォルトでは左側にギュッと詰まってしまいます。これをどう配置するか決めるのが `justify-content` プロパティです。

- **`justify-content: center;`**：要素を**中央揃え**にします。（Aboutセクションのアイコンと文字を中央に寄せたいときなどに便利！）
- **`justify-content: space-between;`**：最初と最後の要素を両端にピタッとくっつけ、間の要素を**均等な間隔**で並べます。

横に並んだ要素たちの「高さ」がバラバラなとき、上下のどの位置で揃えるかを決めるのが `align-items` プロパティです。

- **`align-items: center;`**：要素を上下の真ん中（中央）で綺麗に揃えます。

横並びにした要素同士がピタッとくっついていると見栄えが悪いです。これまでは `margin` を使って調整していましたが、Flexboxではもっと簡単にすき間を空けられる **`gap`（ギャップ）** という便利なプロパティがあります。

```css
.likes-container {
  display: flex;
  gap: 20px; /* 横並びになった子要素同士の間に、一発で20pxのすき間を作ります */
}
```
