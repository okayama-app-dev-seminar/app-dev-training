# 05.課題提出

総集編です。

1. mainブランチから新たにgitというブランチを切ってください.
2. 各課題のスクリーショットを課題番号をファイル名にして, このディレクトリに保存してください.
3. 「gitに関する課題の提出」というメッセージで全てのスクショをコミットしてください.
4. 05_rebaseで作成したコミットは削除せずそのまま提出してください
5. 自身のGithubリポジトリにローカルの変更履歴とブランチを同期させてください.
6. Github上で自分のmainブランチに対してプルリクエストを作成してください.
7. プルリクエストのURLをDiscordの課題提出チャンネルへ共有してください.

## おまけ(次の課題を始める前に)

プルリクエストが通ったら, リモートのmainブランチへマージしましょう.  
その後, リモートのmainブランチをローカルのmainブランチに反映させてください.  
また, もう必要ないgit, git-practice, git-conflict-practiceというブランチを削除しましょう.
